Class AddressReferencedAsset<T>

A AddressReferencedAsset is a class that's used for referencing assets ingame.

The asset referenced can either be a direct reference, or a reference via an address

Some AddressReferencedAsset<T> can load their assets via catalog, such as AddressReferencedItemDef

An AddressReferencedAsset<T> has implicit operators for casting to it's T Type, and for casting into a boolean. It also has implicit operators for encapsulating a string and T inside an AddressReferencedAsset<T>

Asset

The Asset of type T contained by this AddressReferencedAsset

Get behaviour


If the asset has not been loaded or is null, the asset is loaded synchronously via addressables.

In the scenario where CanLoadFromCatalog is set to true, the asset itself is null, and Initialized is false, a warning message is displayed, this is because the AddressReferencedAsset instance can use an asset name directly by loading it via the game's catalogues.

Set Behaviour


Setting this property sets the address to null if the provided value is not null.
