Class AddressReferencedFamilyDirectorCardCategorySelection

A AddressReferencedAsset is a class that's used for referencing assets ingame.

The asset referenced can either be a direct reference, or a reference via an address

Some AddressReferencedAsset<T> can load their assets via catalog, such as AddressReferencedItemDef

An AddressReferencedAsset<T> has implicit operators for casting to it's T Type, and for casting into a boolean. It also has implicit operators for encapsulating a string and T inside an AddressReferencedAsset<T>

T is RoR2.FamilyDirectorCardCategorySelection
