Class AddressableDCCSPool

A AddressableDCCSPool is a version of a RoR2.DccsPool that can be created from the editor itself, it allows you to create complex DccsPools using R2API's AddressReferencedAsset system and your own existing spawn cards
You should also see AddressableDirectorCardCategorySelection
