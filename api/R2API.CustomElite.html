Class CustomElite

Class that defines a custom Elite type for use in the game All Elites consist of an RoR2.EliteDef, an RoR2.EquipmentDef and a RoR2.BuffDef. An Elite can only spawn if its in one of the available RoR2.CombatDirector.eliteTiers (vanilla or custom). Please check the constructors docs for more information.

CustomElite(EliteDef?, IEnumerable<EliteTierDef>)

You can omit giving a value to RoR2.EliteDef.eliteIndex, as it'll be filled in automatically by the game. For your custom elite to spawn, you need to provide an enumerable of RoR2.CombatDirector.EliteTierDef as second parameter. The API will then add your EliteDef in them. You can also make a totally new tier, by either directly modifying the array through GetCombatDirectorEliteTiers() and OverrideCombatDirectorEliteTiers(EliteTierDef?[]?) or by using AddCustomEliteTier(EliteTierDef?)
