InteractableActions

Event used to edit the pool of interactables that can spawn in a given stage.

First parameter is the RoR2.ClassicStageInfo.interactableDccsPool, which is used to select a RoR2.DirectorCardCategorySelection.

A RoR2.ClassicStageInfo.interactableDccsPool usually contains, in a vanilla stage, the following : 1 RoR2.DccsPool.poolCategories, written out in DirectorAPI.Helpers.InteractablePoolCategories.

The first category, Standard has, right now, a single RoR2.DccsPool.ConditionalPoolEntry (contained in RoR2.DccsPool.Category.includedIfConditionsMet) which contains the RoR2.DirectorCardCategorySelection used for DLC1 SOTV. Note : This RoR2.DccsPool.ConditionalPoolEntry is only here for stages that were setup for having DLC1 SOTV content. If the RoR2.DccsPool.ConditionalPoolEntry.requiredExpansions is not enabled in the current lobby, they are not added to the pool of choice. Right now, there is a single PoolEntry in RoR2.DccsPool.Category.includedIfNoConditionsMet, which is the vanilla (no expansion) dccs.

MonsterActions

Event used to edit the pool of monsters that can spawn in a given stage.

First parameter is the RoR2.ClassicStageInfo.monsterDccsPool, depending if the stage was updated or not to use the new RoR2.DccsPool, some of the RoR2.DccsPool.Category and RoR2.DccsPool.PoolEntry can be here or not.

For the Artifact of Dissonance RoR2.RoR2Content.Artifacts.mixEnemyArtifactDef, the original DCCS is located in RoR2.RoR2Content.mixEnemyMonsterCards which is represented by the second parameter of this event.

A RoR2.ClassicStageInfo.monsterDccsPool usually contains, in a vanilla stage, the following : 3 RoR2.DccsPool.poolCategories, written out in DirectorAPI.Helpers.MonsterPoolCategories.

Below is an explanation of the RoR2.ClassicStageInfo.monsterDccsPool categories.

The first category, Standard has, right now, a single RoR2.DccsPool.ConditionalPoolEntry (contained in RoR2.DccsPool.Category.includedIfConditionsMet) which contains the RoR2.DirectorCardCategorySelection used for DLC1 SOTV. Note : This RoR2.DccsPool.ConditionalPoolEntry is only here for stages that were setup for having DLC1 SOTV content. If the RoR2.DccsPool.ConditionalPoolEntry.requiredExpansions is not enabled in the current lobby, they are not added to the pool of choice. Right now, there is a single PoolEntry in RoR2.DccsPool.Category.includedIfNoConditionsMet, which is the vanilla (no expansion) dccs.

The second category, Family has, right now, multiple RoR2.DccsPool.ConditionalPoolEntry (contained in RoR2.DccsPool.Category.includedIfConditionsMet) which contains the RoR2.DirectorCardCategorySelections used for family events. They don't have a corresponding RoR2.DccsPool.ConditionalPoolEntry.requiredExpansions, so they are effectively always added to the pool of choice.

The third category, VoidInvasion has, right now, a single RoR2.DccsPool.ConditionalPoolEntry (contained in RoR2.DccsPool.Category.includedIfConditionsMet) which contains the RoR2.DirectorCardCategorySelection used for void invasion events. It has a corresponding RoR2.DccsPool.ConditionalPoolEntry.requiredExpansions, so they are only added to the pool of choice if the DLC1 expansion is enabled. Note : This category is only here if the stage has DLC content, which is not guaranted.
