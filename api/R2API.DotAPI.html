RegisterDotDef(DotDef?, CustomDotBehaviour?, CustomDotVisual?, CustomDotDamageEvaluation?)

customDotBehaviour code will be executed when the dot is added to the target. Please refer to the game AddDot() method for example use case. customDotVisual code will be executed in the FixedUpdate of the DotController. Please refer to the game FixedUpdate() method for example use case. customDotDamageEvaluation code will be executed instead of TakeDamage in the EvaluateDotStacksForType of the DotController.
