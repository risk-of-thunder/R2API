Add(CustomElite?)

Add a custom elite to the elite catalog. If this is called after the RoR2.EliteCatalog inits then this will throw. EliteTierDefs shouldnt be empty if you want to see your custom elite appear in the game. You can also directly modify RoR2.CombatDirector.eliteTiers. Please check the constructors docs of CustomElite for more information.

AddCustomEliteTier(EliteTierDef?)

Allows for adding a new elite tier def to the combat director. Automatically insert the eliteTierDef at the correct position in the array based on its RoR2.CombatDirector.EliteTierDef.costMultiplier When adding a new elite tier, do not fill the eliteTypes field with your custom elites defs if your goal is to add your custom elite in it right after. Because when doing EliteAPI.Add, the API will add your elite to the specified tiers EliteTierDefs.
