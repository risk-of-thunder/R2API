Class ExtendedRuleChoiceDef

Represents an Extended version of a RuleChoiceDef Unlike regular RuleChoiceDefs, an ExtendedRulechoiceDef can have multiple requirements, such as multiple expansions or multiple unlockables required.

Utilize these collections instead of the singular requirements
