Add a custom equipment item to the list of available items. Value for EquipmentDef.equipmentIndex can be ignored. We can't give you the EquipmentIndex anymore in the method return param. Instead use EquipmentCatalog.FindEquipmentIndex after catalog are init If this is called after the EquipmentCatalog inits then this will return false and ignore the custom equipment item.
