Interface INetRequest<TRequest, TReply>

Interface for network requests which will execute OnRequestReceived() when received by the targeted machine(s). Must be used in conjunction with INetRequestReply<TRequest, TReply>"/>. Check ExamplePing for an example implementation.

Must implement Serialize for how to serialize the class / struct through a NetworkWriter and how to deserialize them through a NetworkReader
