Interfaces

INetCommand

Interface for network messages which will execute OnReceived() when received by the targeted machine(s).

INetMessage

Interface for network messages which will execute OnReceived() when received by the targeted machine(s).

Must implement Serialize for how to serialize the class / struct through a NetworkWriter and how to deserialize them through a NetworkReader

INetRequestReply<TRequest, TReply>

Interface for network replies which will execute OnReplyReceived() after the original target received and executed OnRequestReceived(). Must be used in conjunction with INetRequest<TRequest, TReply>"/>. Check ExamplePingReply for an example implementation.

Must implement Serialize for how to serialize the class / struct through a NetworkWriter and how to deserialize them through a NetworkReader

INetRequest<TRequest, TReply>

Interface for network requests which will execute OnRequestReceived() when received by the targeted machine(s). Must be used in conjunction with INetRequestReply<TRequest, TReply>"/>. Check ExamplePing for an example implementation.

Must implement Serialize for how to serialize the class / struct through a NetworkWriter and how to deserialize them through a NetworkReader
