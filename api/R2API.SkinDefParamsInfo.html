Struct SkinDefParamsInfo

A container struct for all SkinDef and SkinDefParams data. Use this to set skinDef values, then call CreateNewSkinDef(). Leave SkinDefParams null to create one automatically with the given arrays. If SkinDefParams is not null, the arrays will be ignored and the SkinDefParams fields will take priority.
