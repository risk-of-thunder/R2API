Parameters
Type Name Description
TeamDef teamDef

The RoR2.TeamDef of the new team.
TeamsAPI.TeamBehavior teamBehavior

Additional configuration not covered by the RoR2.TeamDef
Returns
Type Description
TeamIndex

The RoR2.TeamIndex of the newly added team, or RoR2.TeamIndex.None if the team limit has been reached.
Remarks

If a team has already been added with the same name configured in teamBehavior, then the previously registered team's RoR2.TeamIndex will be returned.

An enumerable for all registered team indices
