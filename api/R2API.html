SoundAPI.Music.MainAndBossTracks

Contains the two MusicTrackDef that are fired by the game music system depending on the game current state. RoR2.MusicTrackDef.states usually have only two states. One AK.Wwise.State should serve a similar purpose as the game one called gameplaySongChoice, which is normally used for telling Wwise which track to play. That gameplaySongChoice AK.Wwise.State should have a custom group id associated with it which should be different than the game one called gameplaySongChoice. The other AK.Wwise.State should have the same group id as the game one which is currently called Music_system and is used for telling the game if its either a Gameplay, a Bossfight, or a Menu track.
